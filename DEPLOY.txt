# FileServer Deployment Instructions

This document outlines the steps to set up, configure, and run the FileServer project on a new Windows environment (such as a Google Cloud VM).

## 1. System Requirements & Installation
First, ensure you have the necessary system components installed. Open PowerShell as Administrator and run the following commands:

```powershell
# Install Google Chrome
winget install Google.Chrome

# Install Node.js
winget install OpenJS.NodeJS

# Install Antigravity IDE (optional)
winget install --id Google.Antigravity --force
```

After installation, verify Node.js is installed by running: `node -v`

## 2. Project Setup
Open a terminal in the root directory of the project (`c:\FileServer`) and install the required dependencies:

```bash
cd c:\FileServer
npm install
```
*Note: This will download all necessary packages defined in `package.json`.*

## 3. Network & Firewall Configuration
Since this server binds to HTTP (80) and HTTPS (443) ports, you must configure firewalls to allow incoming traffic.

**A. Windows Defender Firewall**
Open PowerShell as an Administrator and run:
```powershell
New-NetFirewallRule -DisplayName "FileServer HTTP/HTTPS" -Direction Inbound -LocalPort 80,443 -Protocol TCP -Action Allow
```

**B. Google Cloud Platform (If hosting on GCP)**
If this machine is a Google Cloud VM, you must explicitly allow web traffic from the Google Cloud Console:
1. Go to Compute Engine -> VM instances.
2. Edit your VM instance.
3. Under "Firewalls", check the boxes for:
   - [x] Allow HTTP traffic
   - [x] Allow HTTPS traffic
4. Save the changes.

## 4. Managing the Server
The application is managed by PM2, running in the background. We have added shortcut commands to make this easy.

**Start the server:**
```bash
npm start
```

**Stop the server:**
```bash
npm stop
```

**Restart the server:**
```bash
npm run restart
```

**Check server status:**
```bash
npm run status
```

*Note: The PM2 files are stored locally in `c:\FileServer\pm2` and the `pm2cmd.ps1` script handles elevated command routing.*
